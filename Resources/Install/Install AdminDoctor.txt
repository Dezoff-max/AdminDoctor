AdminDoctor Installation / Установка AdminDoctor
==========================================

English
-------
1. Drag AdminDoctor.app to Applications.
2. Open AdminDoctor from Applications.
3. If macOS blocks this unsigned build, prefer the safer option:
   Control-click AdminDoctor.app, choose Open, then confirm Open.
4. Internal/testing workaround only: if your organization allows temporarily
   disabling Gatekeeper, run:

   sudo spctl --master-disable

   Open AdminDoctor, then re-enable Gatekeeper immediately:

   sudo spctl --master-enable

AdminDoctor does not upload telemetry and does not run sudo commands itself.

Русский
-------
1. Перетащите AdminDoctor.app в папку «Программы».
2. Запустите AdminDoctor из папки «Программы».
3. Если macOS блокирует неподписанную сборку, сначала используйте более
   безопасный вариант: Control-click / правый клик по AdminDoctor.app,
   выберите «Открыть», затем подтвердите «Открыть».
4. Только для внутреннего тестирования: если политика вашей организации
   разрешает временно отключить Gatekeeper, выполните:

   sudo spctl --master-disable

   Откройте AdminDoctor, затем сразу включите Gatekeeper обратно:

   sudo spctl --master-enable

AdminDoctor не отправляет телеметрию и сам не запускает sudo-команды.
